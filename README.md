Author: Habeeb Abdul
License: MIT

## Description
VillageMed AI Symptom Tracker is a real-time facial analysis tool designed for health monitoring. It utilizes computer vision to detect facial landmarks and estimate potential health metrics.

## Features
* Real-time Emotion Detection: Detects states like "Happy/Alert" or "Stressed/Tired".
* Vital Sign Estimation: Simulates temperature, heart rate, and respiratory rate.
* Data Logging: Automatically logs session data to `emotion_log.txt`.
* Screenshot: Press 's' to save analysis frames locally.
